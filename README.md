 **Fluxo de trabalho Git local**](apoia.se/teomewhy "apoia.se/teomewhy")  
1. git checkout -b  
2. cria ou atualiza arquivos  
3. git status  
4. git add *arquivos*  
5. git status  
6. git commit -m "minha mensagem"  
7. git checkout main  
8. git merge nova_branch  
 **Fluxo de trabalho GitHub <> Local (projeto próprio ou da sua empresa)**  
9. git clone  
10. git checkout -b <nova_branch>  
11. alterações de arquivos  
12. git status  
13. git add *arquivos*  
14. git status  
15. git commit -m "nova mensagem"  
16. git push origin <nova_branch>  
17. abrir Pull request no GitHub para main  
18. excluir <nova_branch> origin  
19. git checkout main  
20. git branch -D <nova_branch>  
 **Fluxo de trabalho GitHub <> Local (projetos open-source)**  
21. Fork do projeto para seu próprio github  
22. git clone  
23. git checkout -b <nova_branch>  
24. alterações de arquivos  
25. git status  
26. git add *arquivos*  
27. git status  
28. git commit -m "nova mensagem"  
29. git push origin <nova_branch>  
30. abrir Pull request no GitHub da branch fork para a main do projeto original  
31. excluir <nova_branch> origin  
32. git checkout main  
33. git branch -D <nova_branch>
